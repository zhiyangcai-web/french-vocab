# Listening Source Guide

Ce depot suit un mode **TV5MONDE-first + RFI backup** pour choisir les supports d'ecoute DELF B2.

## 1. TV5MONDE Apprendre le francais

Source par defaut pour la preparation actuelle.

- Niveau cible: B1+, B2, B2+.
- Priorite aux supports avec video, transcription ou sous-titres, exercices et correction.
- Usage principal: entrainement de type examen, comprehension globale, reperage de details, extraction d'expressions, reformulation d'un point de vue.
- Themes prioritaires: societe, education, environnement, travail, sante, numerique, logement, consommation, medias.
- Si la page n'a pas de transcription ou de sous-titres officiels, le support peut rester en ecoute extensive, mais pas en ecoute intensive principale.

## 2. RFI Journal en francais facile

Deuxieme choix, surtout pour l'actualite.

- Priorite aux episodes avec transcription officielle.
- Usage principal: vitesse de journal, vocabulaire public, endurance auditive, sujets de societe.
- Si l'audio est trop dense ou trop rapide, marquer le niveau B2+ ou C1- et reduire les taches.

## 3. RFI Savoirs

Source utile pour les dossiers thematiques et les supports pedagogiques.

- Priorite aux pages avec transcription, dossier ou activites.
- Usage principal: approfondir un theme DELF B2 avec une structure plus pedagogique que l'actualite brute.

## 4. Sources supplementaires

Ces sources peuvent completer le systeme, mais ne sont pas le choix par defaut pour le moment.

- 1jour1actu: plutot B1+/B2, utile pour construire le contexte et le lexique de base.
- Franceinfo: B2/B2+, utile pour l'ecoute extensive de l'actualite.
- France Culture / France Inter: B2+/C1-, surtout pour les entretiens longs; a reserver pour une phase ulterieure.

## Copyright et transcript

- Ne pas telecharger ni heberger les fichiers audio ou video.
- Le site conserve uniquement `sourceUrl`.
- Si une transcription officielle existe, conserver `transcriptUrl`.
- Ne pas copier de longs transcripts tiers dans le depot.
- On peut conserver de courtes phrases de travail, des difficultes, des mots cles et les transcripts fournis explicitement par l'utilisateur.
- Sans transcription ou sous-titres officiels, marquer le support comme ecoute extensive.
